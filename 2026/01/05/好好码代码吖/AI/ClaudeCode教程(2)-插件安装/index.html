安装Claude Code插件

安装完VS Code后，现在可以安装Claude Code插件了。

从VS Code市场安装（推荐）

  1. 打开VS Code
  2. 点击左侧边栏的扩展图标（或使用快捷键Ctrl+Shift+X/Cmd+Shift+X）
  3. 在搜索框中输入Claude Code
  4. 在搜索结果中找到Claude Code插件，点击安装按钮
  5. 安装完成后，插件会自动启用

手动安装插件

如果无法从VS Code市场安装插件，可以使用以下方式手动安装：

  1. 访问Claude Code插件的GitHub仓库或官方网站，下载最新的VSIX文件
  2. 打开VS Code
  3. 点击左侧边栏的扩展图标（或使用快捷键Ctrl+Shift+X/Cmd+Shift+X）
  4. 点击扩展面板右上角的更多操作按钮（三个点）
  5. 选择从VSIX安装
  6. 选择下载的VSIX文件，点击安装
  7. 安装完成后，重新加载VS Code以启用插件

验证插件安装

安装完成后，可以通过以下方式验证插件是否成功安装：

  1. 打开VS Code
  2. 点击左侧边栏的Claude Code图标（如果已添加到边栏）
  3. 或者，使用快捷键Ctrl+Shift+P/Cmd+Shift+P打开命令面板
  4. 输入Claude Code，如果看到相关命令，则说明插件已成功安装

插件更新

Claude Code插件会定期更新，提供新功能和修复bug。可以通过以下方式更新插件：

  1. 打开VS Code
  2. 点击左侧边栏的扩展图标（或使用快捷键Ctrl+Shift+X/Cmd+Shift+X）
  3. 点击扩展面板右上角的更多操作按钮（三个点）
  4. 选择检查更新
  5. 如果有更新可用，点击更新按钮即可更新插件

或者，在扩展面板中找到Claude Code插件，如果有更新可用，会显示更新按钮。
